# What This Site Is For

This site is for running AI work in a way that you can review and compare later.

Most people do not come here because they want to chat with a model once. They come here because they want to test a setup, compare models, save a useful workflow, or keep a record of what happened in a run.

The site helps with questions like these:

"Which model gave me the best result for this task?"

"Was the better result worth the extra cost?"

"If I run this again later, can I use the same setup?"

"Can I keep the prompts, documents, settings, and results together in one place?"

That is what the site is for.

In simple terms, you set up a run, choose the input and the models, start the job, and then look at the results. If you want, you can make the run more advanced by adding evaluation, comparison, or combine steps. But the basic idea stays the same.

This site is not mainly a chat app. It is a run-and-review tool.

If you want the shortest path to a real result, read [getting-started.md](getting-started.md).
